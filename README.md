# Font Viewer

Preview font files in an editor in Atom. Support zooming via cmd-+, cmd--, and cmd-0. Based on the [image-view](https://github.com/atom/image-view) package.

Currently supports the following file extensions:

- `.otf`
- `.ttf`
- `.woff`
